# Simple House

## 概要
「インテリアデザイン会社のホームページ」をテーマに制作したWebサイト。

## 特徴
画面設計はRustafied.comを参考にさせていただきました。
画面右上の月アイコンをクリックすると、ダークテーマに切り替えることができます。
サイト内の画像をクリックすると、拡大表示されます。

## 制作時期
2021年6月

## 使用言語・技術
- HTML
- CSS
- JavaScript
- jQuery
